# homebrew-peek

Homebrew tap for [peek](https://github.com/rsdrahat/peek) — a light, native macOS markdown viewer.

## Install

```bash
brew install --cask rsdrahat/peek/peek
```

Or tap first:

```bash
brew tap rsdrahat/peek
brew install --cask peek
```

## Upgrade

```bash
brew upgrade --cask peek
```

New peek releases auto-update this tap via GitHub Actions, typically within a minute of the upstream tag landing.

## Uninstall

```bash
brew uninstall --cask peek
# and to also remove preferences:
brew uninstall --zap --cask peek
```
